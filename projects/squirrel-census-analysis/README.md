# 🐿️ The Great Squirrel Census Data Analysis (with Pandas!)

This project explores and analyzes the 2018 Central Park Squirrel Census dataset using Python and Pandas.

## Features

* Load and process real-world dataset
* Analyze squirrel characteristics (e.g., color distribution)
* Generate simple summaries and insights

## Tech Used

* Python
* Pandas

## What I learned

* Working with CSV files using Pandas
* Basic data analysis and grouping
* Turning raw data into meaningful outputs

## Notes

This project represents my early experience with data analysis using Python and real-world datasets.
